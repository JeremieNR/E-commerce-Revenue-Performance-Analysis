# E-commerce Revenue Performance Analysis
**[This project is still a work in progress]**

This project analyzes e-commerce transaction data from Kaggle to evaluate revenue performance across product categories, individual products, and countries.

An interactive **Power BI dashboard** was developed to showcase key revenue metrics, highlight top-performing categories and products, and visualize geographic revenue distribution. The dashboard includes **slicers, drillthrough functionality, and tooltips**, enabling focused exploration and deeper insights.
